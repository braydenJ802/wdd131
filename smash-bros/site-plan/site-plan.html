Super Smash Bros Characters - Site Plan

Brayden Jones

WDD 131-01

Overview

Purpose

This will be a dynamic website that showcases fighters from Super Smash Bros Ultimate (their move sets, game origin, etc.). This acts as a high-level overview of the game.

Audience

This is for Nintendo fans (like those who enjoy playing Super Smash already) and those who are curious to see what the game is like without having to play!

Dynamic elements

The main page will feature all of the fighters, neatly organized as a grid of clickable images. Upon selection, the individual fighter page corresponding to that fighter will be opened. This will showcase specific deatils about that particular fighter. (This includes their name, weight class, game origin, move sets, perceived "tier", and one example combo). This information will be dynamically pulled from a JSON object.

This uses the following:

  1. Url Parameters
  2. Multiple pages
  3. Dynamic Updates (DOM manipulation, HTML templates, JSON)
  4. Conditional branching

Typography

Heading Font: 'Nunito', 'Helvetica Neue', Arial, sans-serif

Paragraph Font: 'Source Sans 3', 'Segoe UI', Tahoma, sans-serif

Typography Rationale

I selected Nunito and Source Sans 3 as my primary typeface pairing to create an approachable yet polished user experience. Nunito's warm, rounded letterforms bring a friendly, gaming-community feel to headings and navigation elements, while Source Sans 3's clean precision ensures excellent readability for detailed fighter information and longer text passages. Both fonts are specifically designed for digital interfaces and offer outstanding accessibility features, including clear character distinction and high screen reader compatibility. This combination strikes the perfect balance between the casual, welcoming nature of gaming culture and the professional presentation expected in modern web design. The pairing also provides robust performance through Google Fonts' reliable CDN, ensuring consistent typography across all devices and browsers while maintaining fast load times for an optimal user experience.
